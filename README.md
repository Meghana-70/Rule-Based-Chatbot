# Rule-Based Chatbot 

## Project Overview

Rule-Based Chatbot is a web-based chatbot developed using HTML, CSS, and JavaScript. The chatbot interacts with users through predefined rules and responds to basic queries related to Artificial Intelligence, programming concepts, date and time, and general conversation.

## Features

* Handles greetings and small talk
* Answers AI and programming-related questions
* Provides current date and time
* Help command for supported queries
* Interactive web-based chat interface
* Rule-based response generation

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Download or clone the repository.
2. Open `index.html` in a web browser.
3. Type your query in the input box and click **Send**.

## Sample Queries

* Hi
* Hello
* What is AI?
* What is Python?
* Explain Machine Learning
* What is Deep Learning?
* Tell me a joke
* What is the current time?
* What is today's date?
* Help
